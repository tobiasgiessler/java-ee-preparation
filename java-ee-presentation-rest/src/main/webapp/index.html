JAX-RS Introduction

REST

  • Representational State Transfer
  • Resource URIs
  • HTTP Methods
  • Method Idempotence
    • Is a method repeadable, without causing additional side effects?
    • GET - read only
    • POST - creeate object (not idempotent => remeber browser warning)
    • PUT - update object
    • DELETE - delete object
  • REST Responses
  • HATEOAS (Hypermedia As The Engine Of Application State)
    • Responses contains links to other resources
  • The Richardson Maturity Model
    • Level 0: One URI, one HTTP method, Request body contains all the details.
    • Level 1: Multiple URIs for multiple resources, one HTTP method, Request body contains all the details.
    • Level 2: Multiple URIs for multiple resources, multiple HTTP methods and HTTP status codes.
    • Level 3: Level 1 + HATEOAS
